A fórmula COUNT em DAX é usada para contar o número de valores em uma coluna. Aqui estão três exemplos de como você pode usar a função COUNT no Power BI: Exemplo 1: Contagem de Vendas Totais Imagine que você tem uma tabela chamada Vendas com uma coluna ValorVenda. Para contar o número total de vendas registradas, você pode usar a seguinte fórmula DAX: dax ContagemVendas = COUNT(Vendas[ValorVenda]) Exemplo 2: Contagem de Produtos Vendidos Suponha que você tenha uma tabela chamada ProdutosVendidos com uma coluna ProdutoID. Para contar o número total de produtos vendidos, a fórmula seria: dax ContagemProdutosVendidos = COUNT(ProdutosVendidos[ProdutoID]) Exemplo 3: Contagem de Registros de Custos Considere uma tabela chamada Custos com uma coluna CustoOperacional. Para contar o número de registros de custos, você pode usar: dax ContagemCustos = COUNT(Custos[CustoOperacional])
