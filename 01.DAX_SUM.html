A fórmula SUM em DAX é usada para somar todos os valores em uma coluna específica. Aqui estão três exemplos de como você pode usar a função SUM no Power BI: Exemplo 1: Soma de Vendas Totais Imagine que você tem uma tabela chamada Vendas com uma coluna ValorVenda. Para calcular o total de vendas, você pode usar a seguinte fórmula DAX: dax TotalVendas = SUM(Vendas[ValorVenda]) Exemplo 2: Soma de Quantidade de Produtos Vendidos Suponha que você tenha uma tabela chamada ProdutosVendidos com uma coluna Quantidade. Para obter a soma total das quantidades de produtos vendidos, a fórmula seria: dax TotalQuantidadeVendida = SUM(ProdutosVendidos[Quantidade]) Exemplo 3: Soma de Custos Operacionais Considere uma tabela chamada Custos com uma coluna CustoOperacional. Para calcular o total dos custos operacionais, você pode usar: dax TotalCustosOperacionais = SUM(Custos[CustoOperacional]) Esses exemplos ilustram como a função SUM pode ser aplicada para agregar dados em diferentes contextos dentro do Power BI.
