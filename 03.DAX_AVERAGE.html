A fórmula AVERAGE em DAX é usada para calcular a média dos valores em uma coluna específica. Aqui estão três exemplos de como você pode usar a função AVERAGE no Power BI: Exemplo 1: Média de Vendas Totais Imagine que você tem uma tabela chamada Vendas com uma coluna ValorVenda. Para calcular a média das vendas, você pode usar a seguinte fórmula DAX: dax MediaVendas = AVERAGE(Vendas[ValorVenda]) Exemplo 2: Média de Quantidade de Produtos Vendidos Suponha que você tenha uma tabela chamada ProdutosVendidos com uma coluna Quantidade. Para obter a média das quantidades de produtos vendidos, a fórmula seria: dax MediaQuantidadeVendida = AVERAGE(ProdutosVendidos[Quantidade]) Exemplo 3: Média de Custos Operacionais Considere uma tabela chamada Custos com uma coluna CustoOperacional. Para calcular a média dos custos operacionais, você pode usar: dax MediaCustosOperacionais = AVERAGE(Custos[CustoOperacional])
