A fórmula MIN em DAX é usada para encontrar o valor mínimo em uma coluna específica. Aqui estão três exemplos de como você pode usar a função MIN no Power BI: Exemplo 1: Mínimo de Vendas Totais Imagine que você tem uma tabela chamada Vendas com uma coluna ValorVenda. Para encontrar o valor mínimo de vendas, você pode usar a seguinte fórmula DAX: dax MinimoVendas = MIN(Vendas[ValorVenda]) Exemplo 2: Mínimo de Quantidade de Produtos Vendidos Suponha que você tenha uma tabela chamada ProdutosVendidos com uma coluna Quantidade. Para obter a quantidade mínima de produtos vendidos, a fórmula seria: dax MinimoQuantidadeVendida = MIN(ProdutosVendidos[Quantidade]) Exemplo 3: Mínimo de Custos Operacionais Considere uma tabela chamada Custos com uma coluna CustoOperacional. Para encontrar o custo operacional mínimo, você pode usar: dax MinimoCustosOperacionais = MIN(Custos[CustoOperacional])
